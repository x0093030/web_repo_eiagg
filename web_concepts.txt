##################### section 07 #################
CSS (Cascading Style Sheets)

Group -> apply to both selectors
    selector , selector {
        color: white;
    }

Child  -> apply to a direct child of left side
    selector > selector{
        color:white;
    }

Descendant -> Apply to a descendent of ;eft side.
    selector selector{
        color: white;
    }

Positioning
    Static, current location.
    Relative, relative to its suposed location or nearest positioned ancestor.
    Absolute, top left corner of webpage.
        Z Index, determine if it is diplayed infront or in the background.
    Fixed,  top left corner of webpage.

##################### VS code shortcuts #################

Fold folds the innermost uncollapsed region at the cursor:
    Ctrl + Shift + [ on Windows and Linux

Unfold unfolds the collapsed region at the cursor:
    Ctrl + Shift + ] on Windows and Linux

Fold All folds all regions in the editor:
    Ctrl + K, Ctrl + 0 (zero) on Windows and Linux

Unfold All unfolds all regions in the editor:
    Ctrl + K, Ctrl + J on Windows and Linux
